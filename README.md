# Projeto-Django

Nome: João Paulo

Formação: Ciências da Computação

Qual projeto penso em desenvolver: Atualmente penso em uma plataforma web para recarga de carteirinhas de ônibus.

Motivo do projeto:
Quero desenvolver esse sistema para facilitar a recarga de carteirinhas de ônibus de forma online, evitando filas e tornando o processo mais rápido e acessível.

## Funcionalidades já adicionadas
- Página inicial do sistema
- Tela de cadastro de usuário
- Tela de login com email e senha
- Logout do usuário
- Dashboard do usuário
- Cadastro de cartão de ônibus
- Exibição dos cartões vinculados ao usuário logado
- Exibição do saldo do cartão
- Tela de recarga
- Recarga simulada
- Atualização do saldo após recarga
- Histórico de recargas
- Exclusão de cartões
- Models criadas para Usuário, Cartão e Recarga
- Registro das models no painel administrativo do Django
- Painel administrativo funcionando para cadastrar dados de teste
